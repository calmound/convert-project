## 1. Unit Converter（通用单位换算工具）

该工具用于在一个统一界面内完成各类物理与工程单位之间的换算，覆盖长度、重量、面积、体积、速度、压力等常见单位体系。用户无需判断单位所属类别，只需选择输入值和目标单位即可立即获得结果，适用于学习、工程计算、日常估算等多种场景。

工具还将内置常用换算对快捷入口、公制与英制体系说明以及高频单位对照表，使其不仅能给出结果，也能帮助用户理解不同单位体系之间的关系，提升复用价值。

## 2. Temperature Conversion Tool（温度换算与参考工具）

该工具专注于不同温标之间的换算，包括摄氏度、华氏度、开尔文等，并针对温度这一高频、强心智场景提供独立页面。用户可以快速完成温度数值转换，满足天气、健康、烹饪、科研等常见需求。

除基础换算外，工具将提供常见温度参考区间（如人体体温、沸点、冰点）及温标使用背景说明，帮助用户判断结果是否合理，而不仅是“算出一个数”。

## 3. Fraction / Decimal / Percentage Tool（分数、小数与百分比转换计算器）

该工具用于在分数、小数和百分比之间进行双向转换，是教育与学习场景中需求极高的一类计算工具。用户可以输入任意一种数值形式，系统将自动输出其他等价表示。

工具会展示必要的中间计算步骤和自动化简结果，解决用户“结果对不对、是怎么来的”的核心疑问，使其在教学、作业核对和考试复习中具备不可替代性。

## 4. Number Base Converter（进制转换工具）

该工具支持二进制、八进制、十进制和十六进制之间的相互转换，面向程序开发、计算机学习及工程计算等场景。用户可自由选择输入进制和目标进制，快速得到准确结果。

在基础换算之上，工具还将提供位宽控制、常见进制说明以及数值表示差异提示，帮助用户理解不同进制在实际系统中的用途，而不仅是完成一次性转换。

## 5. File Size Calculator & Download Time Tool（文件大小与传输时间计算器）

该工具不仅用于文件大小单位之间的换算（bit、byte、KB、MB、GB、TB 等），还用于帮助用户理解文件在真实使用场景中的影响。用户可以输入文件大小并选择网络速度，直接计算下载或上传所需时间。

同时，工具将提供二进制与十进制单位差异说明、存储容量估算（例如某容量硬盘可存放多少文件），解决用户在存储和传输判断上的常见困惑。

工具将支持基本的分辨率、音质和文件体积控制，并清晰提示转换前后的变化，帮助用户在质量与体积之间做出合理取舍。
